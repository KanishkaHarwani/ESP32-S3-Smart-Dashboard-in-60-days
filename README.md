# ESP32-S3 Smart Dashboard in 60 Days

> A beginner-to-professional embedded systems project that builds a production-style IoT dashboard while teaching modern firmware development, FreeRTOS, networking, software architecture, and embedded C++.

---

## About

This repository documents my journey of learning embedded systems by building a feature-rich Smart Dashboard using the ESP32-S3.

Rather than following disconnected tutorials, this project follows a structured 60-day roadmap where every concept is immediately applied to a real firmware project.

The goal is to learn not only how to use the ESP32, but also how professional embedded software is designed, implemented, tested, and documented.

---

## Project Goals

* Learn embedded systems from the ground up
* Build a production-style firmware project
* Understand modern embedded software architecture
* Learn FreeRTOS and real-time programming
* Work with networking and web APIs
* Practice professional Git and GitHub workflows
* Create a portfolio-quality open-source project
